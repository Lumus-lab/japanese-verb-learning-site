# 日文動詞變化規則總整理

日文動詞變化看似複雜，但其實只要先「判斷動詞的分類」**，接著套用**「對應的變化公式」，就能輕鬆掌握！

## 第一部分：動詞的三大分類

在進行任何變化之前，我們必須先辨認該動詞屬於哪一個群組（分類）。

### 1. Ⅲ群動詞（不規則動詞／カ變、サ變）

數量最少，必須直接背誦的動詞。

- **来る（くる）**：來（カ行變格）
    
- **する**：做（サ行變格）。包含所有「名詞 + する」的動作，如：勉強する、実習する、コピーする。
    

### 2. Ⅱ群動詞（上下一段動詞）

特徵是字尾一定是「る」**，且「る」前面的那個音在五十音的**「い段（上一段）」**或**「え段（下一段）」。

- **上一段**：起(お)きる、見(み)る、降(お)りる
    
- **下一段**：寝(ね)る、食(た)べる、教(おし)える
    

### 3. Ⅰ群動詞（五段動詞）

只要**不是**上述兩類的動詞，全部都是Ⅰ群動詞。

特徵是字尾落在五十音的「う段」（う、く、ぐ、す、つ、ぬ、ぶ、む、る）。

- 例如：買(か)う、書(か)く、話(はな)す、立(た)つ、飲(の)む、行(い)く。
    
- **⚠️ 陷阱（例外五段動詞）**：
    
    有些動詞字尾是「る」，且前面是「い/え段」，看起來像Ⅱ群，但其實是Ⅰ群五段動詞！
    
    _口訣特徵：兩個平假名以上，共用一個漢字（如：帰る=かえる），或是常見的例外單字。_
    
    - **常見例外**：帰(かえ)る、入(はい)る、走(はし)る、知(し)る、切(き)る、要(い)る、減(へ)る、蹴(け)る、散(ち)る。
        

## 第二部分：動詞的七大變化（以五十音段落為基礎）

日文動詞有七種基本型態，五段動詞的變化完全依循五十音的「あ・い・う・え・お」五個段落來改變字尾。

### ① 第一變化：未然形（ない形 / 否定形）

用於表示否定（不～）。

- **Ⅰ群（五段）**：將字尾の「う段」音改成「あ段」音 ＋ **ない**。
    
    - _注意：字尾如果是「う」，要改成「わ」而不是「あ」。_
        
    - 行(い)**く** $\rightarrow$ 行(い)**か**ない
        
    - 買(か)**う** $\rightarrow$ 買(か)**わ**ない (例外)
        
- **Ⅱ群（一段）**：去掉字尾「る」 ＋ **ない**。
    
    - 食(た)べる $\rightarrow$ 食(た)べ**ない**
        
- **Ⅲ群（不規則）**：
    
    - 来る $\rightarrow$ **こ**ない
        
    - する $\rightarrow$ **し**ない
        

### ② 第二變化：連用形（ます形 / たい形）

用於禮貌語氣（～ます）、想做某事（～たい）、一邊...一邊...（～ながら）。

- **Ⅰ群（五段）**：將字尾の「う段」音改成「い段」音 ＋ **ます / たい**。
    
    - 行(い)**く** $\rightarrow$ 行(い)**き**ます
        
    - 飲(の)**む** $\rightarrow$ 飲(の)**み**たい
        
- **Ⅱ群（一段）**：去掉字尾「る」 ＋ **ます / たい**。
    
    - 起(お)きる $\rightarrow$ 起(お)き**ます**
        
- **Ⅲ群（不規則）**：
    
    - 来る $\rightarrow$ **き**ます
        
    - する $\rightarrow$ **し**ます
        

### ③ 第三變化：終止形（辞書形 / 原形）

就是字典上查得到的原形。用於句尾結尾、或是加上「な」表示強烈禁止。

- **全群動詞**：維持原樣（字尾為「う段」）。
    
    - 行く。 （去。）
        
    - 行くな。（不准去！）
        

### ④ 第四變化：連体形（接名詞 / 時 / ので）

用來修飾後面的名詞。外觀上與「第三變化（辞書形）」**一模一樣**。

- **全群動詞**：維持辞書形原樣 ＋ 名詞 / 時 / ので。
    
    - 日本へ**行く**時、... (去日本的時候...)
        
    - **食べる**もの (可以吃的東西)
        

### ⑤ 第五變化：仮定形（ば形 / 條件形）

用於表示假設、條件（如果～的話）。

- **Ⅰ群（五段）**：將字尾の「う段」音改成「え段」音 ＋ **ば**。
    
    - 行(い)**く** $\rightarrow$ 行(い)**け**ば
        
    - 買(か)**う** $\rightarrow$ 買(か)**え**ば
        
- **Ⅱ群（一段）**：去掉字尾「る」 ＋ **れば**。
    
    - 食(た)べる $\rightarrow$ 食(た)べ**れば**
        
- **Ⅲ群（不規則）**：
    
    - 来る $\rightarrow$ **く**れば
        
    - する $\rightarrow$ **す**れば
        

### ⑥ 第六變化：命令形

用於強烈命令（去！吃！做！）。語氣較粗魯，常使用於軍隊、比賽加油、男性口語。

- **Ⅰ群（五段）**：將字尾の「う段」音改成「え段」音（不加任何字）。
    
    - 行(い)**く** $\rightarrow$ 行(い)**け** (去！)
        
    - 飲(の)**む** $\rightarrow$ 飲(の)**め** (喝！)
        
- **Ⅱ群（一段）**：去掉字尾「る」 ＋ **ろ**。
    
    - 食(た)べる $\rightarrow$ 食(た)べ**ろ** (吃！)
        
    - 起(お)きる $\rightarrow$ 起(お)き**ろ** (起床！)
        
- **Ⅲ群（不規則）**：
    
    - 来る $\rightarrow$ **こい**
        
    - する $\rightarrow$ **しろ** (或 せよ)
        

### ⑦ 第七變化：意向形（意志形 / 意量形）

用於提議（～吧！）、或是表達自己的打算（意向形 + と思います）。可以視為「～ましょう」的普通體。

- **Ⅰ群（五段）**：將字尾の「う段」音改成「お段」音 ＋ **う**。
    
    - 行(い)**く** $\rightarrow$ 行(い)**こ**う
        
    - 買(か)**う** $\rightarrow$ 買(か)**お**う
        
- **Ⅱ群（一段）**：去掉字尾「る」 ＋ **よう**。
    
    - 食(た)べる $\rightarrow$ 食(た)べ**よう**
        
- **Ⅲ群（不規則）**：
    
    - 来る $\rightarrow$ **こ**よう
        
    - する $\rightarrow$ **し**よう
        

## 第三部分：特殊的「連用形」—— て形 / た形 (音便)

「て形」用於祈求(～てください)、現在進行式(～ています)。

「た形」用於過去式(～た)、經驗(～たことがあります)。

這兩種變化的規則一模一樣！**Ⅰ群（五段）動詞需要發生「音便」（發音變化）**。

### Ⅱ群（一段）與 Ⅲ群（不規則）

非常簡單，直接跟「ます形」一樣，把「ます」換成「て」或「た」。

- 食(た)べる $\rightarrow$ 食(た)べ**て** / 食(た)べ**た**
    
- する $\rightarrow$ し**て** / し**た**
    
- 来る $\rightarrow$ き**て** / き**た**
    

### Ⅰ群（五段動詞）的音便規則大解密 🌟

五段動詞要根據「原形字尾」是哪一個假名，來決定如何變化：

|   |   |   |   |   |
|---|---|---|---|---|
|**原形字尾**|**音便類型**|**變化方式**|**例單字**|**變化後 (て形 / た形)**|
|**う、つ、る**|**促音便**|變成 **「って」/「った」**|買(か)う<br><br>  <br><br>立(た)つ<br><br>  <br><br>帰(かえ)る|買**って** / 買**った**<br><br>  <br><br>立**って** / 立**った**<br><br>  <br><br>帰**って** / 帰**った**|
|**む、ぶ、ぬ**|**撥音便**|變成 **「んで」/「んだ」**|飲(の)む<br><br>  <br><br>遊(あそ)ぶ<br><br>  <br><br>死(し)ぬ|飲**んで** / 飲**んだ**<br><br>  <br><br>遊**んで** / 遊**んだ**<br><br>  <br><br>死**んで** / 死**んだ**|
|**く、ぐ**|**イ音便**|變成 **「いて」/「いた」**<br><br>  <br><br>(若為ぐ，則為 いで/いだ)|書(か)く<br><br>  <br><br>急(いそ)ぐ|書**いて** / 書**いた**<br><br>  <br><br>急**いで** / 急**いだ**|
|**す**|**無音便**|變成 **「して」/「した」**|話(はな)す<br><br>  <br><br>出(だ)す|話**して** / 話**した**<br><br>  <br><br>出**して** / 出**した**|

**⚠️ 唯一死角（超級例外）：**

- **行(い)く**：字尾雖然是「く」，但它**不發生イ音便，而是促音便**！
    
    - 行(い)く $\rightarrow$ $\times$行(い)いて $\rightarrow$ $\bigcirc$ **行(い)って** / **行(い)った**
